# Email Automation Tool 📧

A Python tool that reads contacts from a CSV file, generates personalized outreach emails using Claude AI, and saves them to a new CSV file.

## What it does

- Reads contact details from `contacts.csv`
- Generates personalized emails for each contact using Claude AI
- Saves all generated emails to `generated_emails.csv`

## Tools and Technologies

- Python
- Claude AI (Anthropic)
- pandas
- python-dotenv
- CSV

## Setup

1. Clone the repo
2. Create a `.env` file with your `ANTHROPIC_API_KEY`
3. Run `pip install anthropic pandas python-dotenv`
4. Add your contacts to `contacts.csv`
5. Run `python email_automation.py`
